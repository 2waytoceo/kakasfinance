Live API mode — how it works
This console fires real HTTP requests to apiconnect.axisbank.co.in when valid credentials are provided. Fill in your Client ID, Client Secret, and OAuth Bearer Token in each module's Params tab.

Leave token blank → sandbox mock (safe for testing).   Add Client ID only → mock 200 OK.   Add a valid Bearer token → live attempt to Axis Bank gateway. CORS-blocked calls fall back gracefully to mock with an explanatory note.
